## 1. 题目

[23. 合并K个升序链表](https://leetcode.cn/problems/merge-k-sorted-lists/)

## 2. 思路

用分治的方法进行两两合并。

```java
public ListNode mergeKLists(ListNode[] lists) {
    return mergeKList(lists, 0, lists.length-1);
}

// 归并排序 分而治之
public ListNode mergeKList(ListNode[] lists, int l, int r) {
    // 终止条件
    if (l == r) {
        return lists[l];
    }
    if (l > r) {
        return null;
    }
    int mid = (l + r) >> 1;
    // 递归
    ListNode l1 = mergeKList(lists, l, mid);
    ListNode l2 = mergeKList(lists, mid + 1, r);
    // 合并
    return mergeTwoLists(l1, l2);
}

// 合并两个链表
private ListNode mergeTwoLists(ListNode list1, ListNode list2) {
    if (list1 == null) {
        return list2;
    }
    if (list2 == null) {
        return list1;
    }
    ListNode dummy = new ListNode(0);
    ListNode node = dummy;
    while (list1 != null || list2 != null) {
        int v1 = list1 == null ? Integer.MAX_VALUE : list1.val;
        int v2 = list2 == null ? Integer.MAX_VALUE : list2.val;
        if (v1 > v2) {
            node.next = list2;
            list2 = list2.next;
        } else {
            node.next = list1;
            list1 = list1.next;
        }
        node = node.next;
    }
    return dummy.next;
}
```
